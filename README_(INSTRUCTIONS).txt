1. copiati repositoria : git clone https://github.com/saddsuppreme/EBS-ENTRY-TEST-TASK
2. copiati comanda : pip install virtualenv
3. copiati comanda : env\Scripts\activate.bat
4. pentru instalare daca -> pip install -r requirements.txt nu lucreaza sunt bibliotecile 
    1. pip install django
    2. pip install djangorestframework
    3. pip install psycopg2
5. copiati comanda : python manage.py migrate -> pentru efectuarea migratiilor
5. copiati comanda : python manage.py runserver
